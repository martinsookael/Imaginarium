Imaginarium is a designplatform for Community Tools.

Installation:
- move to sites/all/themes/
- enable and mark as default in Themes
- unenable "cmtls" from Sections
- move /imagimodule to sites/all/modules/imagimodule and enable it.

Imaginarium takes functionality from Cmtls_theme. 
If You want to make Your own design, declare Imaginarium as parent for future updates.

For designers:
Change the backgrounddesign from page.tpl.php and the colors from edit_me.css.
